# r-rhizogene

## Installation

Install the package

```
devtools::install_github("yasoomn/r-rhizogene")
```


## Style

Color palette included

Rhizogene light

![](./light_palette.png)

## Usage

Load the package

```
lbrary(rhizogene)
```
Use the rhizogene theme by default.

```
theme_set(theme_rhizogene())
```
